# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of ACA-Py TypeScript client
- Connection management (create, receive, accept, delete invitations)
- Schema creation and retrieval
- Credential definition creation and retrieval
- Credential issuance workflow (offer, issue, store)
- Proof request and verification workflow
- Wallet DID management
- Basic messaging support
- Agent status checks
- Comprehensive TypeScript type definitions
- Custom error classes for different operation types
- Full test coverage with Jest
- GitHub Packages publishing configuration
- Detailed documentation and examples
